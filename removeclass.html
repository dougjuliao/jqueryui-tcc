RemoveClass

Com esse efeito é possível adicionar uma classe a algum elemento do código.

O código para adicionar a classe seria assim: 


            $( ".addclass-botao" ).click(function() {
              $( ".box-class" ).addClass( "nomedaclass");
              return false;
            });

Para complementar colocarei o removeClass do jQuery para o exemplo ficar mais dinâmico.

O código ficaria assim: 


            $( ".removeclass-botao" ).click(function() {
              $( ".box-class" ).removeClass( "nomedaclass" );
            });
