"Fique a vontade para navegar por aqui, esse é um site com bastante conteúdo de aprendizado sobre o jQuery UI."

A ideia desse projeto é mostrar Uma única página web que carrega o resto de seu conteúdo por ajax, esse método também é conhecido como Single Page Application e além de mostrar sobre esse método Irei mostrar diversos efeitos da biblioteca javascript jQuery UI(User Interface).
Espero que todos se divirtam conhecendo essa fascinante biblioteca e seus diversos efeitos

Linguagens e competencias utilizadas: html5,css3,javaScript,jQuery e jQuery UI.

Autor do projeto: Douglas Julião

API de documentação: jQuery UI

Tema TCC: Demonstração do jQuery UI utilizando Single Page Application

Curso Superior: Sistemas de Informação

Instituição: Universidade Anhanguera
