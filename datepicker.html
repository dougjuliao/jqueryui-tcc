Datepicker

Esse efeito nos permite escolher uma data para ser colocada no input, é um calendário do jquery ui.

Para usar o datepicker é preciso adicionar esse código: 


                $('.datepicker-input').datepicker();

Data2: Esse é o datepicker com display button

Data3: Esse é o datepicker com display de mês e ano

Data4: Esse é o datepicker com display Multiplo

Data5: Esse é o datepicker com Icone
