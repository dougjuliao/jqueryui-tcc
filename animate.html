Animate

Esse efeito é bem parecido com o addClass, ele altera os atributos de um elemento com enfase em cores e tamanho.

O código para adicionar esse efeito seria assim: 


                $(function() {
                    var state = true;
                    $( ".botao-animate" ).click(function() {
                      if ( state ) {
                        $( ".box-class" ).animate({
                          backgroundColor: "#fff",
                          color: "#000",
                          width: 500
                        }, 1000 );
                      } else {
                        $( ".box-class" ).animate({
                          backgroundColor: "#000",
                          color: "#fff",
                          width: 700
                        }, 1000 );
                      }
                      state = !state;
                    });
                  });
