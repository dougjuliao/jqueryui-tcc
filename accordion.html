Accordion

Efeito accordion do jQuery UI - Abas selecionáveis para mostrar e esconder conteúdo.

O código default do accordeon ficaria assim: 


                 $('#id-seletor').accordion();

Clique nos botões abaixo para ver o efeito.

  • Default
  • Collapse
  • Customize icons
  • Fill Space
  • No auto hight
  • Open Hover
  • Sortable

Esse é o accordion default

Aqui nós só temos o efeito accordion do jQuery UI default, é o básico e utilizando a biblioteca só é preciso chamar o efeito com esse código: 

 $('#id-seletor').accordion();

Esse é o accordion collapse

Esse efeito permite que todas as abas fiquem fechadas, podendo clicar na aba aberta e fechar, para adicionar esse efeito precisamos colocar a opção collapsible como true. 

                $('#id-seletor').accordion({
                    collapsible:true
                });

Esse é o accordion Customize icons

Esse efeito permite modificar o icone das abas ou retirá-los, no css do jquery ui encontramos muitos icones usados, para criar seu próprio icone é preciso utilizá-lo com o css. 

            $('#id-seletor').accordion({
                icons: { "header": "ui-icon-plus", "activeHeader": "ui-icon-minus" }
            });

Esse é o accordion Fill Space

Esse efeito permite que o accordion seja redimensionado, para adicionar esse efeito precisamos colocar a opção heightStyle como "fill","auto" ou "content": 

                    $('#id-seletor').accordion({
                        heightStyle:"fill"
                    });
                    $( "#accordion-resize" ).resizable({
                        minHeight: 140,
                        minWidth: 200,
                        resize: function() {
                            $( "#accordion" ).accordion( "refresh" );
                        }
                    });

Esse é o accordion sortable

Esse efeito permite a alteração da ordem das abas, para implementar esse efeito basta usar esse codigo: 

                $("#accordion-sortable").accordion({
                    header: "> div > h3"
                 })
                 .sortable({
                    axis: "y",
                    handle: "h3",
                    stop: function( event, ui ) {
                        ui.item.children( "h3" ).triggerHandler( "focusout" );
                    }
                 });
