Droppable

Esse efeito faz com que seja possível arrastar itens para dentro de algum content definido.

O código default do droppable ficaria assim: 


                $( ".drag-drop" ).draggable();  // draggable para arrastar ate o content
                $( ".drag-drop-default" ).droppable({ // drop pré definido pelo jquery ui
                drop: function( event, ui ) {
                    $( this )
                      .addClass( "ui-state-highlight" )
                      .find( "p" )
                        .html( "Dropped!" );
                  }
                });
